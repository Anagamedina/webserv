◆ shouldCloseConnection()

bool HttpRequest::shouldCloseConnection ( ) const

Verifica si se debe cerrar la conexión con el cliente después de enviar la respuesta.

Note
HTTP/1.1: Por defecto es keep-alive (persistente) Solo cerramos si explícitamente dice "close" HTTP/1.0: Por defecto se cierra después de cada respuesta Solo mantenemos abierta si explícitamente dice "keep-alive" HTTP_VERSION_UNKNOWN o versión no soportada: cerrar por seguridad
