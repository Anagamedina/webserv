Detailed Description

Represents the configuration for a specific location within a server block.

Handles parsing and storing settings defined in a 'location block', such as:

  • content root directory
  • allowed HTTP methods (GET, POST, DELETE, HEAD)
  • default index files in a vector
  • autoindex status boolean
  • file upload directory
  • HTTP redirection
  • CGI handlers like a map
